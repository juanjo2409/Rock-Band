# 🍽️ Fusión Gourmet Festival

Landing Page responsive desarrollada con HTML5, CSS3 y JavaScript para un festival gastronómico de vanguardia.

## 🌐 Demo
https://tuusuario.github.io/proyectorestaurante

---

## 📌 Descripción

Proyecto tipo landing page enfocado en:

- Diseño moderno y responsive
- Menú hamburguesa para desktop y móvil
- Hero section visual e impactante
- Tabla de horarios de catas
- Sección informativa del festival
- Galería multimedia
- Video promocional
- Footer profesional

---

## 🛠 Tecnologías

- HTML5
- CSS3
- JavaScript
- Flexbox
- CSS Grid
- Responsive Design
